Overview

This represents a collection of permissions on allowed on a shared file or folder.

This is an example of a serialized MemberActionList:

[{
  "action": {
    ".tag": "remove"
  },
  "allow": false,
  "reason": {
    ".tag": "target_is_self"
  }
}, {
  "action": {
    ".tag": "make_owner"
  },
  "allow": false,
  "reason": {
    ".tag": "target_is_self"
  }
}]
